## Requirements

* JDK version 7+

### Compile and run tests

```
./gradlew clean build test
```

### IDE Setup

#### Intellij

```
./gradlew cleanIdea idea
```

#### Eclipse

```
./gradlew cleanEclipse eclipse
```

### Run the application
```
./gradlew bootRun
```

In a browser go to `http://localhost:8080/`.
